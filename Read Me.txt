Step 1:
    - Create a database in MySQL.
    - Change the database connection configuration in the "mysql_db_schema.js" file.
Step 2:
    - Change the CLIENT_URL in the "server/.env" file to the IP for socket connection.
    - Change the NEXT_PUBLIC_API_URL in the "frontend/.env" file to the IP for socket connection.
Step 3:
    - In the terminal, run "npm i" in both directories.
Step 4:
    - To start the server, run the command "npm start".
    - To start the frontend, run the command "npm run dev".
